              # 问题4：应用闪退

              • 解决方案1：本地打包指定版本后，运行安装
              cd example
flutter build apk --debug --target-platform android-arm
adb install -r build/app/outputs/apk/debug/app-debug.apk
              1
              2
              3
              • 解决方案2：打开example/android的目录工程，执行运行，此时会做gradle sync库，需要科学上网，或通过修改工程目录下的gradle配置并 修改/Users/song/flutter/packages/flutter_tools/gradle（根据安装flutter目录设置）下的flutter.gradle文件和resolve_dependencies.gradle文件，参考flutter gradle配置。修改完成后进行gradle sync，成功后直接运行项目。
