# 操作步骤

  • 在main工程下index.html修改路由连接
  • 在main工程下的index.js中注册微应用
  • 在子应用中修改入口文件，增加mount bootstrap unmount方法
  • 在子应用中引入_webpack_public-path__= window.INJECTED_PUBLIC_PATHBY_QIANKUN 变量
  • 修改子应用的root元素id，不能与其他子应用冲突
  • 如果用到路由，需要修改路由的basepath配置，与main工程下注册的跟路径保持一致
  • 修改子应用的webpack配置，{library: '应用名', libraryTarget: 'umd'}
  • 修改子应用的webpackDevServer配置增加header:{'access-control-allow-origin': '*'}
